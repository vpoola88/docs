Request A New Font

If the font desired for a specific athlete is not available among the list in the theme settings, follow the below to aquire and submit a new font.

  • 1. Search & download the .ttf version of the font online.
  • 2. Convert the .ttf version of the font to a .otf version (use everythingfonts.com/ttf-to-otf
  • 3. Submit .otf to the developers
