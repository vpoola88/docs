In order to complete the site launch, the developers will need a Google Meta Tag for site verification. To do this:,

  • 1. Log into Webmaster Tools with access.
  • 2. Select or add the website you wish to verify.
  • 3. Select alternative method, and supply the developers with the HTML Tag method output
  • 4. After the developers have imported the meta tag, select Verify at the bottom to finish site verification
  • 5. *NOTE: If another google/anthro email trys to verify the site, it will not verify. Each account is given a unique Meta Goolge Tag, so that same account is the one that must check verification in the future
