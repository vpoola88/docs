Gallery

To create a gallery/galleries for a new athlete:

  • 1. Log into Flickr and navigate to the Photostream option.
  • 2. Create a New Album.
  • 3. Give the New Album a Name and drag the images into the album.
  • 4. Once the album is created, select "Batch Edit", then select "Add Tag" to give every image a very unique ID tag.
  • 5. To add the gallery onto the website, navigate to the backend and create the page "Me" (or whatever gallery is relevant). In the "Content" box, select the "<>" tags for html, pase in the tag used in Flickr.
  • 6. To select the cover photo, upload an image that is 900x600 to photobucket and provide developers with the image link.

Video

To create a video gallery for a new athlete:

  • 1. Log into Vimeo and upload the athlete video
  • 2. Select Done and Go to Video
  • 3. Scroll down and add the .myshopify.com and live domain URL information into seperate rows.
  • 4. Select the Embed option for the video. On this page, select the Preset dropdown and set it to "anthro Private". Scroll down and save.
  • 5. Select Videos at the top menu and create a group. Give the group the website name and add any videos related to that group.
  • 6. Select Save and submit the top link to the developers to add for videos
  • 7. To select the cover photo, upload an image that is 900x600 to photobucket and provide developers with the image link
