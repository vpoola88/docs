When a new athlete is created, a new newsletter key must be generated. To do this, follow the steps below:

  • 1. Log into Sendy: Sendy Link.
  • 2. Add a new Brand for the new athlete. Enter in the "New Brand" information like so, ignoring all other settings input.
  • 3. Select "New Athlete" from the Brand list, and then select "View all lists" form the left hand navigation.
  • 4. Add a New list title "Newsletter". One added, select "View all lists" again from the left hand navigation to view the newsletter ID.
  • 5. Click on edit --> in subscribe success page section, enter in url where user will be redirected after signing up for a newsletter. Scroll down and save.
  • 6. Copy the ID field for the newsletter, and paste this in the Sendy field for the athlete.
