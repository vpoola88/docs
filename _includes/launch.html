To activate the athlete website,
  • 1. Visit the admin panel in shopify and navigate to Settings, Domain. Select Add Existing domain at the top and fill out the domain without www first.
  • 2. Read the instructions carefully. Copy the listed IP to edit the A record.
  • 3. To get the DNS records, log into netfirms (or whatever the domain registration is) and navigate to DomainCentral
  • 4. Navigate to the domain and select the domain to open its sub-menus.
  • 5. Select the option labeled DNS.
  • 6. In the Modify drop down, select the A record. From the instructions, paste in the IP address for the Points To: field for the A Record. IF the website is NOT a site transfer, delete all information in the A record except the athlete url (markuswheaton.com) and the value * (means all). Both records should point to the specified shopify IP.
  • 7. Return to the domain panel in shopify and select Done. Then select Add Existing Domain, and fill in the www portion of the url. Copy the listed url to edit the CNAME record for the DNS information.
  • 8. In the DNS dropdown, navigate the CNAME. Delete all information and only have www route to the shopify url of the athlete.
  • 9. In the Shopify Domain panel, select done. The site will now take time to populate. Revisit the domain settings in Shopify and see all options listed ask "Ok" to verify the site has completed launch.
  • 10. Once the site has been moved to our Shopify Plus account (will see a shopify plus badge in the admin panel or be notified from Christian), return to the DNS information for the CNAME records. Change the shopify url to the shopify SSL url. ie: markus-wheaton.myshopify.com --> ssl.shopify.com.
