B2B Platorm

Setting up a B2b Site will vary across each project. The following steps are an overall scope of how to help development with B2B projects and are to help provide a stepwise developmental process.

1. SiteMap Layout

Development time is expedited when an overall sitemap is provided to better understand the responsabilities of each page. Provide a sitemap with reference to the following link. Creating one in powerpoint or using a word document is helfpul. SiteMap Reference

2. Google Docs Content Layout

After a sitemap is created, the next step would be to divy content into google docs for the developers. Create a singel google doc per page, and place in any content for the team to copy over. If content is not available, use placeholder content so the developers can continue with page layouts.

3. Image Asset Creation

Simultaneous to content layout, it is crucial that assets are cut and provided for the first phase of development. To do so,

  • 1. Get all assets from design.
  • 2. In photobucket, create an album for the project. The Create New Album option is on the left side navigation.
  • 3. Create sub-albums for images that will go on each page. These sub-albums will help narrow the location of the image as well as avoid duplicates. To create a sub album, select the main album on the left hand navigation and then select the option Create sub-album on the right hand Album Actions menu.
  • 4. Provide development with the photobucket links that will go into the google docs for where content is placed. This link is located in the right hand menu under Links to share this photo. Use the link labeled Direct. When you click on this field, the link will be automatically copied to your clip board.

Image Compression

Often times when the website is put on a test server, the images (mainly for banners) will load slower the first time before the site is cached. To help prevent this, compress the images so the img file size itself is more optimal for the web.

  • To compress the image, use the following link and follow the steps displayed on the website. JPEG-OPTIMIZER

CSV Uploads

If a project (i.e. Mahaska) will have data that needs to be listed out, use the CSV tool to provide the developers the data in a proper format. To do so,:

  • visit: Anthro CSV and follow the steps displayed on the website.
